    Na podstawie otrzymanego adresu URL podstrony samochodu, np: www.otomoto.pl/oferta/volkswagen-tiguan-life-2-0-tdi-150-km-od-reki-plichta-gdansk-ID6EJroo.html, pobierana jest strona HTML i zapisywana w @page. Pobierane z tej strony są następujące dane: @model : String - nazwa modelu samochodu @data : Hash - zbiera wszystkie dane zawarte w części szczegóły na podstronie otomoto @price : String - zawiera cenę samochodu

    # File car.rb, line 17
def initialize(car_page)
  @url = car_page
  @page = Nokogiri::HTML(URI.open(@url))
  @model = self.fetch_car_model
  @data = Hash.new
  @price = self.fetch_car_price
  self.fetch_car_data
end

def fetch_car_data
  @page.css("li.offer-params__item").each do |item|
      @data.store(item.css(".offer-params__label").text.strip, item.css(".offer-params__value").text.strip)
  end
end
